ECOTECNOLOGÍA en dam

Proyecto del 21 de Abril de 2017

Proyecto general

Desarrollamos un proyecto basado en el módulo de sistemas informáticos orientado a la ecología con recursos reutilizables, almacenando la información en recursos web y cálculos de propia cosecha

Materiales reciclados

El proyecto esta realizado con materiales reutilizables como cartón y piezas de electrónica antiguas

Bases ecológicas

Calculamos el indicador de impacto ambiental generado por la demanda humana en los ecosistemas del planeta.

Fase 1

Documentación

  • Grupo 1

    Este grupo de trabajo se encargó de informarse sobre posibles ideas para que el hardware(diseño externo) fuera algo llamativo y relacionado con la ecología(decorado) a la par de que siguiera manteniendo la esencia del hardware propiamente dicho.

  • Grupo 2

    El segundo grupo se encarga del desarrollo de esta web de información general mas la web especializada en calcular los niveles de gases nocivos en el medio ambiente(huella ecológica) requiriendo una tarea de investigación sobre esos mismos datos.

  • Grupo 3

    Grupo el cual se encarga del montaje de periféricos, es decir, herramientas personalizadas para el hardware que se utilizará informándose de las mejores opciones de montaje para este proyecto.

    • Fase 2

    Desarrollo

  • Grupo 1

    El grupo hace grandes avances en la planificación de esta parte del proyecto del como se van a organizar y como van a trabajar: Hemos traido una torre antigua, desmontándola, además de montar un ratón artesanalmente con un cristal. Luego se pasará a hacer otro tipo de modificacionnes que nombraremos más adelante.

  • Grupo 2

    El grupo en esta fase desarrolla la estructura de la página web planificando de como van a ir la cabecera, el cuerpo y el pie, además se trabaja en añadirle complementos

      • Grupo 3

        Usamos un raspberry pi para introducirle un sistema operativo y luego añadirle los juegos, acto seguido cogeremos un dispositivo para poder jugar a dichos juegos.

      • Fase 3

      Montaje

    • Grupo 1

      El primer grupo en este punto comenzó el desmontaje de una torre antigua para coger sus piezas y hacer un pc reciclado al cual se le han metido modificaciones como ampliación de RAM, meterle el sistema operativo que soporta en este caso el windows xp entre otras cosas(De forma alternativa este mismo grupo trabaja en el diseño de carteles).

      • Grupo 2

        El montaje del grupo 2 consistió en hacer la web base que es en la que estamos presentes y rellenarla con información variada del proyecto.
        También este grupo se encargó de hacer el cartel de presentación del proyecto con piezas recicladas.

    • Grupo 3

      Realizan una instalacion de una distribucion de raspberri pi que se llama retropie
